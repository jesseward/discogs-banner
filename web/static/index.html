About Discogs Collage

Enter your Discogs account name, wait 2 minutes for the image to process then save your custom album art collage. Please note that your Discogs collection must be marked as public and have at least 20 items present.

You can find the source code or report any bugs at the Github page. Please note that due to a limit on the Discogs image API, sporadic errors may appear when we reach 1000 image downloads per day. You're free to try again after a 24 hour period.
